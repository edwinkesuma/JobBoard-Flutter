# JobBoard Mobile

JobBoard Mobile is a Flutter application that allows users to browse job listings and apply for jobs posted by companies.

The app connects to the JobNest Backend REST API built with Spring Boot.

## Features

- User registration and login
- Browse job listings
- View job details
- Apply for jobs
- Company job management
- Clean UI built with Flutter

## Tech Stack

- Flutter
- Dart
- REST API integration
- Dio HTTP client
- Bloc / State Management

## Screens

Main screens:

- Login
- Register
- Job List
- Job Detail
- Apply Job
